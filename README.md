# Web ESPTool

A web-based tool for flashing firmware to ESP8266/ESP32 microcontrollers using the WebSerial API.

## Tech Stack

- Vue 3 + TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
pnpm install
pnpm dev
```

## Browser Support

Requires a browser with WebSerial API support (Chrome, Edge, Opera).
